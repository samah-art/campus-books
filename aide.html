Comment s'inscrire ?

  1. Cliquez sur les trois barres du menu en haut a gauche de la page.
  2. Sélectionnez “Se connecter” dans le menu.
  3. Cliquez sur “S’inscrire” (si vous n’avez pas encore de compte).
  4. Vous serez redirigé vers le formulaire d’inscription.
  5. Remplissez tous les champs demandés :
    • Nom
    • Prenom
    • Email
    • Numero de telephone
    • Mot de passe
    • Genre
    • Niveau d'etude
  6. Cliquez sur “inscrire” pour valider votre compte.

Comment passer commande?

  1. Cliquez sur les trois barres du menu en haut a gauche de la page.
  2. Sélectionnez “Tous les Livres” dans le menu.
  3. Ajoutez tous vos livre dans votre panier grace au bouton "ajouter au panier".
  4. Cliquer sur l'icone du panier (les livres) en haut a droite.
  5. Cliquez sur "Valider ma commande"
  6. Avant de confirmer la commande :
    • Si vous etes deconnecté connecter vous
    • Si vous avez pas de compte, créez en un via la page d'inscription
  7. Cliquez sur “confirmer la commande” pour valider votre compte.

Comment vendre mes ressources?

  1. Cliquez sur les trois barres du menu en haut a gauche de la page.
  2. Sélectionnez “Vendre” dans le menu.
  3. Remplissez les champs demandés:
    • Titre du cours
    • Prix du cours
    • Joignez une photo de ce cours
  4. Cliquez sur “mettre en vente” pour le mettre en vente.
  5. Nous vous enverrons un email dés que votre cours est acheté
